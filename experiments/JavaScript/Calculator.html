In this experiment, I've designed a simple calculator using Javascript and CSS.
 
Inorder to get the layout of a simple calculator, I've set the width of the calculator space to 325px as shown below:
 
#calc {
    width: 325px;
    height: auto;
}
 
And, to get the C button and the screen of top the keys, I've put them in a separate <div> tag and to differentiate the screen from the rest of the calculator, I have 
 
set the background color of the screen to a different value.
 
The hover background color for the keys are set as below:
 
.keys span:hover {
background: #9c89f6;
}
 
.keys span.eval:hover {
background: #abb850;
}
 
.top span.clear:hover {
background: #9c89f6;
}
 
Now, let's move on to the JavaScript part of the experiment.
 
To get the values of the keyevent, I use:
 
document.querySelectorAll('#calc span');
 
Next, iterating in a loop for the length of the key, if the '=' key is selected, the value is calculated and the value is set in the screen.
 
The code illustrating this part is:
 
for (var i = 0; i < keys.length; i++) {
    keys[i].onclick = function (e) {
 
        var input = document.querySelector('.screen');
        var inputVal = input.innerHTML;
        var btnVal = this.innerHTML;
 
 
        if (btnVal == 'C') {
            location.reload();
        }
 
        if (btnVal == '=') {
            var equation = inputVal;
 
            if (equation)
                input.innerHTML = eval(equation);
 
        }
 
        else {
            input.innerHTML += btnVal;
        }
 
    }
}
