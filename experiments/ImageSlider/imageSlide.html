Image slider effect


In this experiment, I have used jQuery and CSS to create an image sliding effect.

The HTML contains the images enclosed in <div>. 

        <div id="slideshow">
            <div id="slideshowWindow">
 
                <div class="slide">
                    <img src="amazon.jpg" class="image" />
                </div>
 
                <div class="slide">
                    <img src="walmart.jpg" class="image" />
                </div>
 
                <div class="slide">
                    <img src="ebay.jpeg" class="image" />
                </div>
 
            </div>
        </div>
 
Next, in the CSS, we can set width and height of the images in the <div>.In the outer <div> the overflow: hidden is set to create the sliding effect. In the inner <div>, the float is set to left. Also, the max-height and the max-width of the image are set to 100% to fit the image within the <div>.
 
#slideshow #slideshowWindow {
width:500px;
height:257px;
margin:0;
padding:0;
position:relative;
overflow:hidden;
}
 
#slideshow #slideshowWindow .slide {
margin:0;
padding:0;
width:500px; 
height:257px;
float:left;
position:relative;
}
 
.image{
    max-width:100%;
    max-height:100%;
    margin: auto;
    position: absolute;
    top: 0; left: 0; bottom: 0; right: 0;
}
