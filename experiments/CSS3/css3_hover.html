In this experiment, I've tried using CSS3 to create an image hover effect.

The ul-design class is used to set the margins, the initial image height and width. This is can be noticed in the classes .ul-design and .ul-design li. The top margin is set to 50 px while the rest of the margins are set to 0px. The image is aligned to the center and displayed as a block. The width and the height of the image are set to 400 px.

The code snippet for this is as follows.

        .ul-design {
            margin: 50px 0 0 0;
            padding: 0;
            list-style: none;
            display: block;
            text-align: center;
            width: 100%;
        }
 
        .ul-design li {
            width: 400px;
            height: 400px;
            display: inline-block;
            margin: 20px;
        }
 
The 'transform' in the .div-item: hover and .div-info bring in the transformation effect upon hover. The Scale is used to control the radius of the transformation and the opacity determines the opacity factor of the transformation.
 
The code snippet for this is:
 
        .div-item:hover .div-info {
            transform: scale(0.95);
            opacity: 1;
        }
 
Upon hover, during the transformation, the .div-info and .div-info h3 provide the styles for the background color and the text displayed during transformation. The transition in .div-info specifies the time for the ease-in-out (for the previous image). I've set the value of this .div-info 0.4s.
 
The code snippet for this is:
 
        .div-info {
            position: absolute;
            background: rgba(63,147,147, 0.8);
            width: inherit;
            height: inherit;
            border-radius: 50%;
            overflow: hidden;
            opacity: 0;
            transition: all 0.4s ease-in-out;
            transform: scale(0);
        }
 
        .div-info h3 {
            color: #fff;
            text-transform: uppercase;
            letter-spacing: 2px;
            font-size: 22px;
            margin: 0 30px;
            padding: 45px 0 0 0;
            height: 140px;
            font-family: 'Open Sans', Arial, sans-serif;
        }
