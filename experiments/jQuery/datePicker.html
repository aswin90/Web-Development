In this experiment, I have explored jQuery Datepicker feature.

I've added the jQuery and jQueryUI. Also, I have added Bootstrap to style the page.

<script src="//code.jquery.com/jquery-1.10.2.js"></script>
<script src="//code.jquery.com/ui/1.11.2/jquery-ui.js"></script>
<link href="//maxcdn.bootstrapcdn.com/bootstrap/3.3.1/css/bootstrap.min.css" rel="stylesheet">
 
I have created a input text box with the id 'datepciker'. Also, I have another read-only field, where the selected date is displayed in the long format.
 
<input type="text" id="datepicker">&nbsp;<input type="text" id="alternate" size="27" readonly>
 
When the text box is clicked, the jQuery function is called. The script is as follows:
 
<script>
        $(function () {
            $("#datepicker").datepicker({
                altField: "#alternate",
                altFormat: "DD, d MM, yy"
            });
        });
</script>
 
 
The datepicker is applied to the text box and a jQuery datepicker is displayed for the user to input the date. The selected date is also displayed in a long format in the nearby read-only box.
