XML Consumption using jQuery

TitleAuthorPrice

In this experiment, I have parsed a XML file using jQuery ajax request, and populate the parsed data in a table with the rows being dynamically generated.

In the <body>, a template for the class is set as follows:

        <table id="chart" class="table">
            <tr><th>Title</th><th>Author</th><th>Price</th></tr>
        </table>
 
In the JS file, when the page loads, the jQuery onload function is called and the ajax request parameters are set as below:
 
                type: "GET",
                url: "/web_resource/books.xml",
                dataType: "xml",
                success: function (xml)
 
In the success function, the XML is parsed and the value of the Title, Author, Price are obtained and set in a new row.
 
For each <book> in the XML, the Title, Author, Price are retrieved and set in a new row.
