This experiment helped me learn the various interactions possible using jQuery.

I've done four interactions, viz:

1. Draggable

2. Resizable

3. Selectable

4. Sortable

Let's take one each and look in detail:

Draggable

The <div> element is as follows:

<div id="draggable">
            <p>Try dragging me!!</p>
        </div>
 
The id is passed to the selector function and the draggable function is called:
 
 $("#draggable").draggable();
 
Resizable
 
The <div> content for Resizable is;
 
  <div id="resizable" class="ui-widget-content">
            <h3 class="ui-widget-header">Resizable</h3>
        </div>
 
The id passed to the selector function and resizable function is called
 
$("#resizable").resizable();
 
Selectable
 
The <div> element of Selectable consists of the following list:
 
 <div id="selectableDiv">
            <ol id="selectable">
                <li class="ui-widget-content">Chelsea</li>
                <li class="ui-widget-content">Southampton</li>
                <li class="ui-widget-content">Aston Villa</li>
                <li class="ui-widget-content">Arsenal</li>
                <li class="ui-widget-content">Swansea City</li>
            </ol>
        </div>
 
This id is passed to the selector function and the selectable() is called:
 
$("#selectable").selectable();
 
Sortable
 
The <div> content for Sortable consists of a list as follows:
 
<div>
            <ul id="sortable">
                <li class="ui-state-default">1</li>
                <li class="ui-state-default">2</li>
                <li class="ui-state-default">3</li>
                <li class="ui-state-default">4</li>
                <li class="ui-state-default">5</li>
                <li class="ui-state-default">6</li>
                <li class="ui-state-default">7</li>
                <li class="ui-state-default">8</li>
                <li class="ui-state-default">9</li>
                <li class="ui-state-default">10</li>
                <li class="ui-state-default">11</li>
                <li class="ui-state-default">12</li>
            </ul>
        </div>
 
The id is passed to the selector function and the following calls are performed:
 
$("#sortable").sortable();
$("#sortable").disableSelection();
