Tooltip Experiment



In this experiment, I have explored jQuery Tooltip feature as I'm planning to use this in my project.

I've added the jQuery and jQueryUI. Also, I have added Bootstrap to style the page.

<script src="//code.jquery.com/jquery-1.10.2.js"></script>
<script src="//code.jquery.com/ui/1.11.2/jquery-ui.js"></script>
<link href="//maxcdn.bootstrapcdn.com/bootstrap/3.3.1/css/bootstrap.min.css" rel="stylesheet">
 
Next, I have created a input text box with the id 'item'. When the cursor is hovered over the input box, the jQuery function is called. This results in the title, being displayed in the Tooltip.
 
<input id="item" class="form-control" title="Enter the item what you want to search" placeholder="Search Item">
 
The script is as follows:
 
<script>
      $(function() {
          $('#item').tooltip({
              track: true
          });
      });
</script>
 
Here, I've set track: true to display the tooltip with the mouse pointer tracking.
