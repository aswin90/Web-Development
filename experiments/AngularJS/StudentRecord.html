In this experiment, I've used AngularJS to display an array of values passed from the script. 
 
I've linked AngularJS and Bootstrap CSS through from CDN to the HTML page as shown below:
 
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.0.1/angular.min.js"></script>
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">
 
In the <body> of the HTML, I've used the ng-app directive to designate the root element of the application.
 
<body ng-app>
 
In the <div> following the <body>, the ng-controller is given the value "StudentRecordController"
 
<div class="container" ng-controller="StudentRecordController">
 
To manipulate the rows for the array, the ng-repeat directive has been used.
 
<tr ng-repeat="student in students">
 
 
Now in the function StudentRecordController the array of students is created and the array is passed to $scope:
 
$scope.students = students;
 
 
For the Add functionality, the values entered in the respective fields are set to a student and the student is pushed to the array
 
     $scope.add = function () {
            console.log($scope.newStudent);
            var student = {
                id: $scope.newStudent.id,
                lastname: $scope.newStudent.lastname,
                firstname: $scope.newStudent.firstname,
                email: $scope.newStudent.email,
            };
            $scope.students.push(student);
        }
 
For the Delete functionality, the index of the row is passed and the row is spliced off from the array
 
 $scope.delete = function (student) {
            var index = $scope.students.indexOf(student);
            $scope.students.splice(index, 1);
        }
 
For the Select functionality, the student is passed to the scope.
 
 $scope.select = function (student) {
            console.log(student);
            $scope.newStudent = student;
        }
