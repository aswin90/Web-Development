In this experiment, I have explored the various form controls in HTML5 such as Text Input, Radio Buttons, Number, Date, Time, Range Slider, Email fields.

The Text Input - Used to get text input from the user. The syntax for the Text Input is as follows:

<input type="text" name="nameval" id="namefield1" required />

Radio Field - Used to get

<input type="radio" name="genderval" id="genderval" value="Male" required />Male

Number - Used to get number input from the user. The syntax for the Number Input is as follows:

<input type="number" name="numval" id="numval" required />

Date Field - Used to get date input from the user. The syntax for the Date Input is as follows:

<input type="date" name="dateval" id="dateval" required />

Time - Used to get Time input from the user. The syntax for the Time Input is as follows:

<input type="time" name="timeval" id="timeval" required />

Range Slider - Used to get a range from the user. The syntax for the Range is as follows:

<input type="range" name="slider" min="0" max="15" onchange="getRangeVal(this.value);" required />

Email - Used to get Email input from the user. The syntax for the Email Input is as follows:

<input type="email" name="emailVal" id="emailVal" required />

 

When the 'Click Here' button is clicked, the values entered are displayed in an alert box. I've used JavaScript for this. The values entered in the form are retrived using document.getElementById.
