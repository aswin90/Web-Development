Light Box Experiment


Click on the image to view in light box mode


Click outside the image to return back to the page



In this experiment, I tried creating a Light Box effect using CSS and jQuery.

The HTML page is a plain page consisting of the images to be displayed.

<div>
 
            <img src="amazon.jpg" class="thumbnail" width="120" onclick="lightbox('<img src=amazon.jpg width=100>');">
            <img src="walmart.jpg" class="thumbnail" width="120" onclick="lightbox('<img src=walmart.jpg width=100>');">
            <img src=" ebay.jpeg" class="thumbnail" width="120" onclick="lightbox('<img src=ebay.jpeg width=100>');">
 
</div>
 
The CSS contains lightbox and lightbox-shadow which define the CSS for displaying the image in Light Box Effect.
 
#lightbox {
position: absolute;
top: 0;
left: 50%;
width: 500px;
margin-left: -250px;
background: #fff;
z-index: 1001;
display: none;
}
 
#lightbox-shadow {
position: absolute;
top: 0;
left: 0;
width: 100%;
height: 100%;
background: #000;
filter: alpha(opacity=75);
opacity: 0.75;
z-index: 1000;
display: none;
}
