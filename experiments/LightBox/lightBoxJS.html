Light Box jQuery


The jQuery plays an important role in acheiving this effect.

On click of an image, <div> with lightbox and lightbox-shadow id's are added to the image as below:

        if ($('#lightbox').size() == 0) {
            var theLightbox = $('<div id="lightbox"/>');
            var theShadow = $('<div id="lightbox-shadow"/>');
            $(theShadow).click(function (e) {
                closeLightbox();
            });
            $('body').append(theShadow);
            $('body').append(theLightbox);
        }
 
Now an AJAX call is made which loads the image with the Light Box effect. While the image is loaded in the Light Box effect, a message 'Loading...' is added to the page.
 
if (ajaxContentUrl != null) {
            
            $('#lightbox').append('<p class="loading">Loading...</p>');
 
            
            $.ajax({
                type: 'GET',
                url: ajaxContentUrl,
                success: function (data) {
                    
                    $('#lightbox').empty();
                    $('#lightbox').append(data);
                },
                error: function () {
                    alert('AJAX Failure!');
                }
            });
        }

To close the Light Box effect, the <div> with the id's are hidden. When the next image, is to be loaded in the Light Box effect, the previous content is cleared and the new image is loaded.

 

function closeLightbox() {
 
    
    (function ($) {
 
        
        $('#lightbox').hide();
        $('#lightbox-shadow').hide();
 
        
        $('#lightbox').empty();
 
    })(jQuery); 
 
}
