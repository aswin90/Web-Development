In this experiment, I have explored the using the Grid View feature. 
 
1. Login with the following usernames - aswin / mani.
 
2. The profile of the particular user is displayed.
 
3. Clicking on 'Add a comment' takes you to the Wall of the user
 
4. Clicking on 'Post a comment' redirects to a page where the Title, Content, Date can be entered and posted to the wall.
 
5. After the comment is added, the Wall with the lastest comments is displayed.
 
In the Login page, when the username is entered, and when we click on 'Login', the username is sent as a query to the Profile page and the profile of that user is loaded.
 
In the Grid view, I have modified the Select query to load the profile of the particular user.
 
When 'Add a comment' is selected, the Wall of the user is displayed. In this page also, Grid view control is used with the Select query modified to retrieve only the comments related to that particular user.
 
When 'Post a comment' is selected, the user can add the Title, Content and Date and Insert the comment.
